# Technologies

This project leverages a modern stack tailored for performance, scalability, and developer experience.

## Core Framework

- **[React 19](https://react.dev/)**: The latest version of the popular UI library.
- **[TypeScript](https://www.typescriptlang.org/)**: For static type checking and improved developer tooling.

## Build & Module Federation

- **[Vite](https://vitejs.dev/)**: A lightning-fast build tool and development server.
- **[@module-federation/vite](https://github.com/module-federation/vite)**: The main library enabling Module Federation for Vite projects.

## State Management

- **[Redux Toolkit](https://redux-toolkit.js.org/)**: The official, opinionated, batteries-included toolset for efficient Redux development.
- **[React Redux](https://react-redux.js.org/)**: Official React bindings for Redux.

## Routing

- **[React Router v7](https://reactrouter.com/)**: Declarative routing for React applications.

## Utilities

- **[@trutoo/event-bus](https://www.npmjs.com/package/@trutoo/event-bus)**: A lightweight event bus for decoupled communication between micro-frontends.
- **[ESLint](https://eslint.org/)**: Pluggable JavaScript linter.

## Package Management

- **npm**: Used for dependency management and running scripts (via `package.json`).
